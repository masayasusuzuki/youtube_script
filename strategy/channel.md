# チャンネル設計

## 基本情報

- **チャンネル名**: じっくりAI駆動開発
- **運営者**: マサ

## コンセプト

非エンジニア向けのAI駆動開発を、じっくり丁寧に解説するチャンネル。

### 使用ツール（特化）
- Cursor
- Claude Code

### ポジショニング
- 広く浅く → ✗
- ニッチに深く → ✗
- **初心者向けにわかりやすく丁寧に → ◎**

### 差別化ポイント
- AIツール解説チャンネルは多いが、開発文脈で初心者向けに丁寧に解説しているチャンネルは少ない
- 運営者自身が非エンジニアとして0→1を経験している
- 「一緒に成長していく」スタンス

## ターゲット視聴者

- 非エンジニアでAI駆動開発を0→1で始めたい人
- 副業志望者
- 社内にエンジニアがいない経営者・担当者
- Xの発信者層より下のレイヤー（初心者）

### ターゲットではない人
- フリーランス志望（レベルが合わない可能性）
- すでにAI駆動開発を実践している中上級者

## 目的・ゴール

1. **収益化** - 教材販売、コンサルへの導線
2. **自己成長** - AIエンジニアとしてのキャリアアップ
3. **スキルの確立** - 独学で身につけたスキルを発信を通じて確かなものにする
